Navigating the Godot Editor üó∫Ô∏è

  • Scene Panel (top-left): Your node hierarchy
  • FileSystem (bottom-left): Project files and assets
  • Viewport (center): Visual editor where you build scenes
  • Inspector (right): Properties of selected nodes
  • Bottom Panel: Output, debugger, and other tools

Think of it like: Scene panel = your object list, Viewport = your canvas, Inspector = your toolbox for each object

Nodes & Scenes: Godot's Building Blocks

Everything in Godot is a Node

  • Node: Basic building block (like a GameObject in Unity)
  • Scene: A saved collection of nodes
  • Hierarchy: Nodes can be parents/children of other nodes

Common Node Types:

  • Node2D - Base for 2D elements with position/rotation/scale
  • Sprite2D - Displays images/textures
  • Polygon2D - Draws custom shapes
  • CollisionShape2D - Defines physics boundaries

Project: Analog Clock üïê

What We'll Build:

  • ‚úì A circular clock face (using Sprite2D with gradient)
  • ‚úì 12 hour indicator wedges (using Polygon2D)
  • ‚úì Three clock arms: hour, minute, second (using Polygon2D)
  • ‚úì Proper parent-child hierarchy for organization

This project teaches you fundamental Godot concepts: scenes, nodes, transforms, and visual elements!

Step 1: Setting Up the Scene

Create Root Node

  1. Click "2D Scene" or add a Node2D
  2. Rename it to "Clock" (F2 or right-click ‚Üí Rename)
  3. This is your root node - everything else will be children
Naming Convention: Use PascalCase for node names
Examples: Clock, HourArm, SecondArm (no spaces)
Remember: The root node represents your entire clock. All parts of the clock will be its children!

Step 2: Creating the Clock Face

Add Face Node

  1. Select "Clock" node ‚Üí Add Child Node (Ctrl+A)
  2. Choose Sprite2D
  3. Rename to "Face"
  4. In Inspector: Texture ‚Üí New GradientTexture2D
  5. Set width and height to 256

Texture Setup:

  • Gradient ‚Üí New Gradient
  • Make right key transparent (alpha = 0)
  • Move left key to 0.98 offset for solid circle

Step 3: Adding Hour Indicators

Create First Indicator

  1. Select "Face" ‚Üí Add Child ‚Üí Polygon2D
  2. Rename to "Hour0"
  3. Set Color to black (#000000)
  4. In viewport: Enable "Select Mode" and "Grid Snap"
  5. Place 3 points to form a triangle at top of face
Duplication Trick: After creating Hour0, duplicate it (Ctrl+D) and rotate by 30¬∞ for each hour indicator. Do this 11 more times!

Step 4: Creating Clock Arms

Build the Arms

  1. Select "Clock" ‚Üí Add Child ‚Üí Polygon2D ‚Üí "HourArm"
  2. Draw a short, thick arrow pointing at Hour0
  3. Give it a counterweight by overlapping the pivot
  4. Duplicate for "MinuteArm" - make it longer/narrower/blue
  5. Duplicate for "SecondArm" - make it sharp/thin/red
Final Hierarchy:
Clock
  ‚Üí Face (Sprite2D)
      ‚Üí Hour0, Hour1, ..., Hour11 (Polygon2D)
  ‚Üí HourArm (Polygon2D)
  ‚Üí MinuteArm (Polygon2D)
  ‚Üí SecondArm (Polygon2D)
